Analisis de editores

Este proyecto analiza tres editores para desarrollo de HTML.
He utilizado los 3 editores citados en la elaboracion de este proyecto

Visual studio code




Visual Studio code es una de las herramientas mas flexibles que encontre para el desarrollo de html.
No obstante cabe destacar que las multiples extensiones y el soporte de otros lenguajes los hacen altamente atractivo.
Dentro de los editores de texto simple me parece el mas completo de los 3.

Sublime text editor




Sublime text editor nos ofrece un workframe mas simple pero igual de eficiente
que Visual Studio code. La unica variante que he encontrado es que Sublime no parece
tan dinamico como Visual studio. No soporta add-ons ni permite la integracion con git.


Notepad++




Notepad++ es uno de los editores de texto "to-go" mas versatiles que he encontrado.
No necesita instalacion alguna y soporta multiples lenguajes.
La ventaja de poder programar en cualquiera de estos lenguajes y luego compilar el
codigo online si fuera necesario lo convierte en mi opinion en uno de los mejores editores.
